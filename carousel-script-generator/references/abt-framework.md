# ABT Storytelling Framework for Carousels

## The Core Formula

**AND** → Set the scene, establish context or a shared truth
**BUT** → Introduce conflict, a problem, or a surprising twist (the emotional trigger)
**THEREFORE** → Deliver the resolution, solution, or key insight

## Why ABT Works in Carousels

Each slide in a carousel is a micro-story beat. The ABT structure maps naturally:

- **Slides 1-2** (Hook + Setup) = the AND — establish the world/context
- **Slides 3-5** (Conflict/Problems) = the BUT — tension, mistakes, obstacles
- **Slides 6-7** (Resolution/Value) = the THEREFORE — solutions, frameworks, answers
- **Slide 7-8** (CTA) = lesson + next action

## ABT Applied Per-Slide

You can also use mini-ABT within individual slides to keep each one engaging:

**Example slide copy:**
"Most creators post every day AND expect growth... BUT consistency without strategy is just noise. THEREFORE: post with intent — one viral-ready post beats seven throwaway ones."

## Carousel-Specific ABT Patterns

### Pattern A: Problem → Solution
- AND: "You want to [goal]"
- BUT: "[Common approach] isn't working / [surprising obstacle]"
- THEREFORE: "Here's the [number]-step framework that actually works"

### Pattern B: Myth → Truth
- AND: "Everyone believes [common wisdom]"
- BUT: "The data shows [surprising contradiction]"
- THEREFORE: "Do [correct approach] instead"

### Pattern C: Story → Lesson
- AND: "[Person/I] was doing [activity]"
- BUT: "Then [unexpected event/realization] happened"
- THEREFORE: "The lesson: [actionable insight]"

### Pattern D: Before → After
- AND: "[Past state] — this was the reality"
- BUT: "One change shifted everything"
- THEREFORE: "Now [improved state] — here's exactly what changed"

## Tension Words (the "BUT" arsenal)
Use these to introduce conflict on problem slides:
- but, however, yet, except, unfortunately
- the problem is, what most people miss, the catch
- here's where it breaks down, the truth is, in reality
- what nobody tells you, the hidden cost, the real reason

## Resolution Words (the "THEREFORE" arsenal)
Use these to deliver payoff on solution slides:
- so here's what works, therefore, the fix is simple
- instead do this, the solution, here's the framework
- this is exactly how, the step-by-step, the key insight
